# Structural Calculation Service LP

This is a static landing page for a structural calculation consulting service.

## Project Structure

- `index.html`: Main landing page structure.
- `style.css`: All conflicting styles, animations, and responsive design rules.
- `script.js`: Interactive elements (Sticky CTA, FAQ accordion, Smooth scroll, Scroll animations).

## Deployment

This project is a static site (HTML/CSS/JS) and can be deployed to any static hosting service.

### Recommended Options:
1.  **Vercel / Netlify**: Simply drag and drop the folder or connect a Git repository.
2.  **GitHub Pages**: Push to a GitHub repository and enable GitHub Pages in settings.
3.  **Standard Web Server**: Upload files via FTP/SFTP to `public_html`.

## Browser Support
- Modern browsers (Chrome, Edge, Safari, Firefox)
- Responsive design supports Mobile (375px+), Tablet, and Desktop.
